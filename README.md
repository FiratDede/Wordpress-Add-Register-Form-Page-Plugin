# Wordpress-Add-Register-Form-Page-Plugin
You can add a register form to your website by using this plugin.
### How to use
Just add  [fd_add_register_form]  shortcode to the page where you want to add the register form page
### Extra Properties
1) When  a user is registered, the user is redirected to home page of the website automatically.
2) If the user is logged in and the user is not admin, the user cannot see the register form page.
3) This plugin adds a settings page to administration menu to control this register form page. An admin or editor can set some features in register form page like minimum username length etc..
